# firebase-telegram-bot

# Configs
```command
$ npm install -g firebase-tools
$ firebase login
$ cd functions
$ npm install
```

## Functions
- [router](https://us-central1-pizzacounterbot.cloudfunctions.net/router)

## Deploy
```command
$ firebase deploy --only functions
```
