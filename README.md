# pizzanhaminhami-telegram-bot

## Technologies
- Firebase
- Javascript

## Configs
```command
$ npm install -g firebase-tools
$ firebase login
$ cd functions
$ npm install
```

## Deploy
```command
$ firebase deploy --only functions
```
